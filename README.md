# Education & Health EDA Project

## Overview
This project performs Exploratory Data Analysis (EDA) on education and waist-AT datasets using Python, Pandas, Matplotlib, and Seaborn. 

## Features
- Summary statistics (mean, median, mode, variance, standard deviation)
- Skewness and kurtosis analysis
- Histograms, boxplots, and density plots
- Q-Q plots
- Scatter plots and correlation analysis

## Files
- `EDA_Analysis.ipynb` : Main notebook
- `education.csv` : Education dataset
- `wc-at.csv` : Waist vs AT dataset

## Instructions
1. Open `EDA_Analysis.ipynb` in Jupyter Notebook or VS Code.
2. Run the cells step by step to view results and plots.
